# Firefox Add-ons 提交信息

## Summary

Copy tab information to clipboard in various formats. Supports 9 formats (URL, Title, Markdown, CSV, JSON, HTML...) and 6 scopes (current tab, window, all windows). Right-click menu and keyboard shortcuts.

将浏览器标签页信息以多种格式复制到剪贴板。支持 9 种格式（URL、标题、Markdown、CSV、JSON、HTML 等）和 6 种范围（当前标签页、当前窗口、所有窗口等）。右键菜单和快捷键操作。

## Description

OmniTabs lets you copy browser tab information to your clipboard in the format you need.

**9 output formats:** URL, Title: URL, Title & URL, Title, Markdown, CSV, JSON, HTML, and HTML Table.

**6 scope options:** Current Tab, Current Window, All Windows, Tabs to the Left, Tabs to the Right, or all tabs Except the Current Tab.

**How to use:**
- Right-click any page, hover over OmniTabs, select a format and scope
- Use keyboard shortcuts: Alt+Shift+C (current tab), Alt+Shift+A (current window), Alt+Shift+W (all windows)
- Click the toolbar icon to set defaults and quick-copy

Includes badge feedback showing copied tab count, and supports English and Chinese (auto-detected from browser language).

Source code: https://github.com/TuTouPower/omni_tabs

OmniTabs 让你以需要的格式复制浏览器标签页信息到剪贴板。

**9 种输出格式：** URL、标题: URL、标题和 URL、标题、Markdown、CSV、JSON、HTML、HTML 表格。

**6 种范围选择：** 当前标签页、当前窗口、所有窗口、左侧标签页、右侧标签页、除当前标签页外。

**使用方式：**
- 右键任意页面，悬停 OmniTabs，选择格式和范围
- 快捷键：Alt+Shift+C（当前标签页）、Alt+Shift+A（当前窗口）、Alt+Shift+W（所有窗口）
- 点击工具栏图标设置默认值并快速复制

复制成功后显示角标反馈（标签页数量），支持中英文双语（自动检测浏览器语言）。

源码：https://github.com/TuTouPower/omni_tabs
